# Pimp My Shell

**Hook:** Configure a shell in 30 seconds.

1. `ssh bhawker@linux.csc.uvic.ca`
2. `git clone https://github.com/hwkr/dotfiles.git .dotfiles`
3. `cd .dotfiles`
4. `./bootstrap`
5. `git checkout server`
6. `cd ../`
7. `source .bashrc`

Toggle Fullscreen: `⌘` `^` `F`  
Toggle Tab Bar: `⌘` `⇧` `F`
